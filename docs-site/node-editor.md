# Node Editor — 可视化量化工作流编辑器

## 概述

Node Editor（节点编辑器）是 Fincept Terminal 中的可视化量化工作流设计工具，允许用户通过拖拽式节点编辑构建复杂的量化交易流水线。该功能与 MCP（Model Context Protocol）工具深度集成，使量化策略的构建、回测和部署变得更加直观高效。

## 核心功能

### 可视化量化工作流

通过图形化界面设计量化策略，将抽象的策略逻辑转化为直观的节点图。用户可以清晰地看到数据流动方向和各个处理环节，便于策略的调试和优化。

### 拖拽式节点编辑

- 无需编程即可构建复杂工作流
- 节点自动对齐，自动连接
- 支持节点的复制、粘贴、撤销和重做
- 实时预览节点执行结果

### MCP 工具集成

与 MCP（Model Context Protocol）协议完全兼容，可调用丰富的外部工具和服务，扩展工作流能力。

## 典型工作流

```
数据获取 → 数据清洗 → 特征工程 → 模型训练 → 回测 → 部署
```

| 阶段 | 说明 |
|------|------|
| **数据获取** | 从多种数据源获取市场数据 |
| **数据清洗** | 处理缺失值、异常值、数据标准化 |
| **特征工程** | 构建技术指标、因子等特征 |
| **模型训练** | 使用机器学习模型训练策略 |
| **回测** | 基于历史数据验证策略有效性 |
| **部署** | 将策略部署到实盘环境 |

## 支持的节点类型

### 数据源节点

- **Polygon** — 专业级实时和历史市场数据
- **Finnhub** — 股票、加密货币、宏观经济数据
- 以及其他 100+ 数据连接器

### 处理节点

- 数据清洗与预处理
- 技术指标计算
- 因子构建

### 模型节点

- 机器学习模型集成
- 因子发现
- 高频交易策略
- 强化学习交易模型

### 回测节点

- 策略性能评估
- 风险指标计算（VaR、Sharpe 等）
- 交易成本模拟

### 脚本集成

- Python 脚本无缝嵌入
- 自定义节点开发
- 第三方库支持
